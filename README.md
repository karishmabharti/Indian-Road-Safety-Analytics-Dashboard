# Indian Road Safety Dashboard

## Executive Summary
This project provides a comprehensive analysis of 20,000 road accidents to drive actionable insights into road safety and traffic management. By employing a "Why, Where, When" analytical framework, the dashboard visualizes accident severity, environmental impacts, and regional risk, enabling stakeholders to make informed decisions for safety interventions.

## Key Insights & Features
* **Location Risk Summary:** Identifies high-risk regions to prioritize infrastructure maintenance and resource allocation.
* **Temporal Analysis:** Visualizes "When" accidents occur (Time-Based Risk) to optimize emergency response dispatch during peak hours.
* **Root Cause Analysis:** Investigates the "Why" behind incidents, correlating driver behavior (distraction, drunk driving) with accident severity.
* **Environmental Impact:** Analyzes the correlation between weather conditions (Fog, Rain, Clear) and road types (Urban vs. Rural vs. Highway).

## Tools & Technologies
* **Visualization:** Power BI (DAX, Calculated Columns/Measures)
* **Data Preparation:** Power Query / Advanced Excel
* **KPIs Tracked:** Total Accidents, Fatality Rate, Accident Severity Index, Average Casualties per Incident.

## Business Impact
This dashboard transforms raw traffic data into a visual intelligence tool, allowing policymakers and road safety committees to pinpoint critical zones and implement targeted interventions that directly aim to reduce the Accident Severity Index.
